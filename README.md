# Инструкция по запуску бота
## Трек - Цифровизация вуза

Данный бот предназначен для получения и отображения расписания занятий студентов и преподавателей, а также основной информации о пользователях образовательного процесса, созданную для MAX Hackathon.

### Требования
- Docker и Docker Compose
- Токен MAX API

Запуск с помощью Docker контейнера (Рекомендуется) 
1. С помощью команды

```
git clone https://github.com/DezzLema/hackathon2025DeadlineYesterday.git
```
Скопируйте проект в любую удобную директорию

2. Откройте Docker Desktop (для Windows/Mac) или Docker Engine (для Linux) и откройте Terminal

3. C помощью команды 

```
cd путь к вашей директории
```
Перейдите в корневую папку с проектом

4. Выполните в Terminal команду

```
docker compose up —build
```
5. Подождите пока проект соберется (в среднем это занимает 5-6 минут)

6. Перейдите в мессенджер Max https://max.ru/t260_hakaton_bot

7. Для начала работы с ботом нажмите кнопку "Начать" или пропишите команду /start

# Запуск через командую строку
## Для корректной работы бота необходимо установить язык Python 3.13+

1. С помощью команды
```
git clone https://github.com/DezzLema/hackathon2025DeadlineYesterday.git
```
Скопируйте проект в любую удобную директорию

2. C помощью команды

```
cd путь к вашей директории
```
Перейдите в корневую папку с проектом

3. В корневой папке проекта выполните следующую команду 

```
pip install -r requirements.txt
```

4. Запустите бота выполнив команду

```
python main.py
```

5. Перейдите в мессенджер Max https://max.ru/t260_hakaton_bot

6. Для начала работы с ботом нажмите кнопку "Начать" или пропишите команду /start
